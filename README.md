# Yasin Rahat's Memo

A production-ready personal website for Yasin Rahat: writing, founder notes,
research, travel stories, podcast conversations, and a private publishing flow.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react
- framer-motion
- Supabase-ready publishing

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Publishing

See `PUBLISHING.md` for the Supabase login and blog publishing setup.
